## BlueVelvet - A Laravel-Friendly PHP Library
To use this with your Laravel 5.0-5.6 project:

```
	"require":           {
		...
		"chaoticwave/blue-velvet": "1.0.*"
		...
	},
```

To use this with your Laravel 5.8 project:

```
	"require":           {
		...
		"chaoticwave/blue-velvet": "1.1.*"
		...
	},
```
